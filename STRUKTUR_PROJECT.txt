🗄️ DATABASE (PostgreSQL)
├── 📦 products
│   ├── id (UUID, Primary Key)
│   ├── name (VARCHAR)
│   ├── price (VARCHAR)
│   ├── category (VARCHAR) → Frame | Sunglasses | Lensa
│   ├── description (TEXT)
│   ├── image_url (TEXT)
│   ├── created_at (TIMESTAMP)
│   └── updated_at (TIMESTAMP)
│
├── ⭐ testimonials
│   ├── id (UUID, Primary Key)
│   ├── name (VARCHAR)
│   ├── text (TEXT)
│   ├── rating (INTEGER) → 1-5
│   └── created_at (TIMESTAMP)
│
├── 📧 contact_messages
│   ├── id (UUID, Primary Key)
│   ├── nama (VARCHAR)
│   ├── email (VARCHAR)
│   ├── pesan (TEXT)
│   ├── status (VARCHAR) → unread | read | replied
│   └── created_at (TIMESTAMP)
│
└── 🔧 services (Optional)
    ├── id (UUID, Primary Key)
    ├── title (VARCHAR)
    ├── description (TEXT)
    ├── icon (VARCHAR)
    ├── features (JSONB)
    ├── created_at (TIMESTAMP)
    └── updated_at (TIMESTAMP)

📁 STORAGE
└── product-images (Bucket)
    ├── Public Access: ✅
    ├── Max Size: 5MB
    └── Formats: PNG, JPG, WEBP

🔐 SECURITY (Row Level Security)
├── Public dapat:
│   ├── SELECT products ✅
│   ├── SELECT testimonials ✅
│   └── INSERT contact_messages ✅
│
└── Authenticated dapat:
    ├── INSERT/UPDATE/DELETE products ✅
    ├── INSERT testimonials ✅
    ├── SELECT/UPDATE contact_messages ✅
    └── Upload/Delete images ✅
